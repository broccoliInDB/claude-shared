# Code Reviewer Memory

이 파일은 code-reviewer 에이전트의 영속 메모리입니다.
리뷰에서 반복 발견되는 패턴을 기록하고, 다음 리뷰에서 우선 체크합니다.

> 이 파일은 프로젝트 로컬에 복사되어 사용됩니다.
> 프로젝트별로 다른 패턴이 축적되며, 2회 이상 반복된 패턴은 /evolve로 팀 규칙 승격을 검토합니다.
